## Overview

This repository hosts the **production build** of my personal portfolio — a single-page, app-router experience that blends WebGL/Canvas visuals with spring-physics motion. It's designed to feel less like a static résumé and more like an interactive product: a cinematic intro, a 3D "book-flip" project gallery, scroll-linked transitions, and a dark, glassmorphic aesthetic throughout.

> The site is deployed via **GitHub Pages** under the `/Portfolio/` base path.

## Highlights

- **Cinematic intro** — animated "THE PORTFOLIO" title sequence with a skip control.
- **3D project & skill cards** — book-flip cards that reveal a scrollable description, tech tags, and an icon on hover.
- **Physics-driven motion** — pointer repulsion, elastic bouncing, and particle fluid mechanics.

## Running Locally

This repo contains the exported static build. Because asset paths are prefixed with `/Portfolio/`, serve it from the parent directory so the base path resolves:

```bash
# Clone
git clone https://github.com/Vibhathcross/Portfolio.git

# Serve from the PARENT of the Portfolio folder
cd Portfolio/..
python3 -m http.server 8000

# Open:
#   http://localhost:8000/Portfolio/index.html
```

Any static file server works (e.g. `npx serve`) as long as the site is reachable under the `/Portfolio/` path.
